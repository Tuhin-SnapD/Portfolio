# Project Images

This directory contains the landing/screenshot images for the portfolio projects.

## Required Images

Add the following images to this directory:

1. **pricepilot-landing.png** - Screenshot of the PricePilot Intelligent Pricing Dashboard
2. **newsmelt-landing.png** - Screenshot of the NewsMelt Advanced News Finder
3. **todos-list-landing.png** - Screenshot of the Todos-List AI-First Task Manager
4. **restaurant-app-landing.png** - Screenshot of the Fullstack Restaurant App
5. **trading-simulator-landing.png** - Screenshot of the Real-Time Stock Trading Simulator
6. **text-summarization-landing.png** - Screenshot of the Text Summarization Models

## Image Guidelines

- **Format**: PNG or JPG
- **Dimensions**: 400x250 pixels (or 16:10 aspect ratio)
- **Quality**: High quality screenshots that showcase the project's main features
- **File Size**: Keep under 500KB for optimal loading

## How to Get Images

1. **Use the HTML Generator**: Open `generate-placeholders.html` in your browser to create placeholder images
2. **From GitHub Repositories**: Check if repositories have a `landing.png` or `screenshot.png` in their root directory
3. **Take Screenshots**: Run the projects locally and take screenshots of the main interface
4. **Create Mockups**: Use design tools to create professional mockups

## Quick Setup

1. Open `generate-placeholders.html` in your browser
2. Click "Download All Images" to download placeholder images
3. Move the downloaded images to this directory (`public/projects/`)
4. Replace with actual project screenshots when available

## Fallback

If images are not available, the portfolio will show a styled placeholder with the project name. The placeholder includes:
- Gradient background matching the portfolio theme
- Project name display
- Professional styling
